## ✍️ Frontend best-practices I follow
- Component-first approach (small, tested units)  
- CSS architecture: utility-first + BEM where appropriate; prefer Bootstrap utilities for speed  
- Accessibility: semantic HTML, keyboard navigation, ARIA attributes where needed  
- Performance: lazy loaded components, code-splitting, optimized assets (images/webp, SVGs)  
- CI: linting, tests, and Storybook/Docs preview on PRs

- create DB + user, set credentials in env vars, install mysqlclient or PyMySQL, run migrations
